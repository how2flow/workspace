# workspace

### script

Run:

```
$ cd workspace
$ ./set --target=<TARGET>
```

TARGET list:

- docker
- opencv
- python3
- ros2
- vi (default)

help:

```
$ ./set --help
```

### docker image

pull image
```
$ docker pull stevejeong/ubuntu:20.04
```

run image
```
$ docker run -it stevejeong/ubuntu:20.04 bash
```
